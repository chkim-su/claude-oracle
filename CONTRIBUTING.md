# Contributing

Pull requests welcome.

## Setup

```bash
git clone https://github.com/n1ira/claude-oracle.git
cd claude-oracle
python -m venv venv
source venv/bin/activate
pip install google-genai requests
```

## Guidelines

- Keep it simple. This is a single-file tool by design.
- Test your changes with both `oracle ask` and `oracle imagine`.
- If adding a new feature, update the README.

## Reporting Issues

Use the issue templates. Include your OS, Python version, and the full error message.
